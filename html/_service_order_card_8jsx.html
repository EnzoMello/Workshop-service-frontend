Detailed Description

Componente de card para exibir um resumo de uma Ordem de Serviço em uma lista.

Author
Enzo Mello

@description Este é um card clicável que mostra informações essenciais de uma OS, como seu número, status e tempo estimado. Ele muda de aparência quando está selecionado.
