Detailed Description

Página principal para o gerenciamento de Boxes da oficina.

Author
Enzo Mello

@description Esta página exibe todos os boxes cadastrados em um layout de cards. Permite ao usuário adicionar novos boxes, navegar para a página de atualização e deletar boxes em lote através de modais interativos.
