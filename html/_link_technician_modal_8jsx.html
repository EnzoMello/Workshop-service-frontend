Detailed Description

Modal complexo para vincular um Técnico e um Box a uma Ordem de Serviço.

Author
Enzo Mello

@description Apresenta duas colunas com listas pesquisáveis, uma para técnicos e outra para boxes, permitindo ao usuário selecionar um item de cada lista para a vinculação.

Parameters
{object}props - Propriedades do componente.
{function}props.onClose - Callback para fechar o modal.
{function}props.onConfirm - Callback executado com os IDs do técnico e do box selecionados.
{Array<object>}[props.technicians=[]] - A lista completa de técnicos disponíveis.
{Array<object>}[props.boxes=[]] - A lista completa de boxes disponíveis.
Returns
{JSX.Element} O modal de vinculação.
