\doxysubsection{Detailed Description}
Componente de card para exibir informações de um Box. 

\begin{DoxyAuthor}{Author}
Enzo Mello
\end{DoxyAuthor}
@description Exibe o identificador e o status de um box. A cor do card muda com base no status (verde para \textquotesingle{}AVAILABLE\textquotesingle{}, vermelho para outros). Reutiliza estilos do Technician\+Card.
