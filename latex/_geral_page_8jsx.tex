\doxysubsection{Detailed Description}
Define a página "{}\+Geral"{}, o painel principal para gerenciamento de Ordens de Serviço (OS). 

\begin{DoxyAuthor}{Author}
Enzo Mello
\end{DoxyAuthor}
@description Esta página renderiza um layout de duas colunas. A coluna da esquerda contém a lista de todas as OS. A coluna da direita exibe, por padrão, cartões de resumo (KPIs e Status dos Técnicos) e, ao selecionar uma OS, exibe os detalhes completos daquela OS. Gerencia todo o estado e as chamadas de API para esta funcionalidade.
